Trabajo Práctico N°8 - Ejercicio 4
Simulación del modelo de Lotka-Volterra
Autores: Astorga Dario - Coria Gaston - Tissera Adrian

PREREQUISITOS:
1-	Tener instalada alguna distribucion de linux. El programa fue escrito, compilado y testeado en
	Ubuntu 12.04

2- 	Tener instalado Gnuplot en el directorio por defecto '/usr/bin/gnuplot'

3- 	El programa fue escrito en Java 8.0 por lo que se recomienda ejecutarlo en esa version
	(No fue testeado en versiones de Java anteriores)

EJECUCIÓN:
Para ejecutar la aplicacion desde la linea de comandos debe encontrarse dentro de la carpeta 
'dist' la cual se encuentra dentro de la carpeta del proyecto y ejecutar el siguiente comando:

java -jar "Simulacion-P08e04.jar"

Nota:

Luego de ejecutar el programa, dentro de la carpeta 'dist/output' se encuentran los gráficos, 
como asi también, los archivos de texto con los datos de salida de la simulación.
